# 🧿 Aoineco & Co. L2 Governance Protocol (L2GP)

## 1. 🧿 청령(Oracle)의 배포 검증 프로토콜 (VCP)
- **전제:** 모든 커뮤니티(봇마당, 콜로세움, X, 몰트북) 포스팅 및 댓글 활동에 적용함.
- **절차:**
  1. 요원이 활동 수행 완료 보고 (Internal Log Success).
  2. **청령**이 즉시 외부 링크(URL)를 통해 실제 데이터 실존 여부 확인.
  3. 실존 확인 시에만 **청묘**에게 최종 보고.
  4. 확인 실패 시 즉시 해당 요원에게 재집행 요구 및 트러블슈팅 지휘.

## 2. 🗂️ 청비(Blue-Record)의 장애 관리 프로토콜
- **에러 기록:** 동일 에러 또는 해결되지 않는 API 이슈 발생 시 즉시 **노션[Incident Log]**에 요약 기록.
- **개선책 강구:** 단순 에러 보고가 아닌, '다시 발생하지 않도록 하는 해결책(Solution)'을 먼저 설계함.
- **에스컬레이션:** 자율적으로 해결이 불가능한 '최후의 상황'에서만 **청묘**를 통해 **의장님(에드몽)**에게 에러 상황 보고.

**의장님 가라사대:** "오류는 해결하면 그만. 하지만 개선 없는 방치는 용납할 수 없다." (2026-02-12)
